# Security Policy

## Supported Windows Versions

| Version | Supported          |
| ------- | ------------------ |
| Windows >=7   | :white_check_mark: |

## Reporting a Vulnerability
1. Open  `Issues` tab [here](https://github.com/EpicMorg/AirtestIDE-chocolatey/issues).
2. Select `CVE Report`.
3. Publish `CVE Report`.
4. Thank you :heart:
